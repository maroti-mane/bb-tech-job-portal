# BB Tech Job Portal

## Project Overview

BB Tech Job Portal is a Recruitment & Placement Management System that connects students, companies, and administrators on a single platform.

This application is being developed as a real-world project using Java Spring Boot and React.js.

## Technology Stack

### Backend
- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate

### Frontend
- React.js
- Vite
- Tailwind CSS

### Database
- MySQL

### Tools
- Git
- GitHub
- Maven
- Docker (Coming Soon)
